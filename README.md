# Road Safety With Us — Client Profile

A single-page company profile to share with clients. Built with Vite (vanilla HTML/CSS/JS).

## Run locally
```bash
npm install
npm run dev      # opens http://localhost:5180
```

## Build for sharing / hosting
```bash
npm run build    # outputs static site to dist/
npm run preview  # preview the production build
```
Deploy the `dist/` folder to any static host (Vercel, Netlify, GitHub Pages).

## Edit content
- Copy, sections, contact details: `index.html`
- Colors, fonts, layout: `src/style.css` (brand tokens at the top)
- Images: `public/images/` (product shots in `categories/`)
